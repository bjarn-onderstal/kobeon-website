# Kobeon-site — lokaal bekijken & feedback geven

Bedankt dat je even meekijkt! Hieronder hoe je de site in een paar minuten lokaal draait.

## Wat je nodig hebt
- **Node.js 18 of hoger** — gratis via https://nodejs.org (kies "LTS"). Na installatie even doorklikken; verder niets nodig.
- Windows. (Mac/Linux kan ook, zie onderaan.)

## Starten (Windows — makkelijkst)
1. Pak de map uit (als je een .zip kreeg).
2. **Dubbelklik op `start-kobeon.cmd`.**
   - De eerste keer installeert hij automatisch de onderdelen (een paar minuten).
   - Daarna opent je browser vanzelf op **http://localhost:3000**.
3. Klaar met kijken? Klik in het zwarte venster en druk **Ctrl+C**, daarna venster sluiten.

> Opent de browser te snel (foutmelding)? Ververs de pagina na een paar seconden — de server start dan net op.

## Starten (handmatig / Mac / Linux)
Open een terminal in deze map en draai:
```bash
npm install      # alleen de eerste keer
npm run dev      # daarna start de site op http://localhost:3000
```

## Wat kun je bekijken
De homepage en deze pagina's (klik door via het menu):
- **Home** (`/`) — hero, "Wat we bouwen", diensten, projecten, sectoren, FAQ
- **Diensten** (`/diensten`) · **Projecten** (`/projecten`) · **Sectoren** (`/sectoren` + de 6 sectorpagina's)
- **Werkwijze** (`/werkwijze`) · **Over ons → Werken bij** (`/werken-bij`, `/vacatures`)
- **Quickscan** (`/quickscan`) · **Discovery-sessie** (`/discovery-sessie`)

## Goed om te weten (nog niet af)
- **Beelden zijn nog placeholders**: de "schermpjes" in de site zijn gecodeerde mini-demo's en de tegels gebruiken merk-kleurvlakken. Echte foto's/screenshots komen later — die hoef je niet te beoordelen.
- **Teksten/cijfers** zijn deels placeholder (bv. testimonials, sommige cijfers).
- Het gaat dus vooral om **opbouw, structuur, toon en gevoel**.

## Waar we feedback op zoeken
- Klopt de **boodschap en toon**? (zakelijk/enterprise, "je/jouw")
- Is de **opbouw** logisch en de navigatie helder?
- Wat **springt eruit** of mist er nog?
- Werkt alles op jouw scherm (desktop én mobiel)?

Noteer je opmerkingen gerust per pagina — alle feedback is welkom. Dank!
